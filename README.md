# add-shebangs 
[![npm](https://img.shields.io/npm/v/add-shebangs)](https://www.npmjs.com/package/add-shebangs)

> A simple CLI tool that adds a shebang command to your file 

## Install
```
npm install -global add-shebangs
```

## Usage
```
shebangs
```
